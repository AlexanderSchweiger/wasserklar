CSV hochladen

Erwartetes Format
  • CSV mit Semikolon-Trennzeichen, Zeichensatz UTF-8
  • Zahlen im österreichischen Format (Komma als Dezimaltrennzeichen, z. B. -1 577,94)
  • Datum im Format DD.MM.YYYY
  • Im nächsten Schritt können Sie die Spalten zuordnen.
Beispielspalten: 
Konto;Datum;KST;KTR;Name;Beschreibung;Steuer;Ausgaben;Einnahmen
Girokonto;02.01.2025;Postgebühr;;Post;Wasserzähler retour;20;-38,82;
Girokonto;02.01.2025;Instandhaltung;Rohrbruch;Huber;Arbeitsleistung;;;500,00
KST = Kostenstelle (Konto), KTR = Projekt, Konto = reales Bankkonto. Fehlende Konten, Projekte und Bankkonten werden automatisch angelegt. Kunden werden per Namensähnlichkeit automatisch zugeordnet.
