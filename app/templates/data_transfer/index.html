Export

Erstellt eine ZIP-Datei mit allen ausgewaehlten Tabellen im dialektneutralen JSON-Format. Die Datei kann spaeter auf einem anderen Wasserklar-System (OSS oder SaaS-Tenant) importiert werden.

  • Stammdaten
  • Buchungen (mit Jahresfilter)
  • Mahnwesen
  • Einstellungen
  • Optional: PDF-Anhaenge
Export starten

Import

Liest eine zuvor erstellte Wasserklar-Export-Datei (.zip) ein. Vor dem endgueltigen Anwenden wird eine Vorschau angezeigt — der Import laeuft transaktional, bei einem Fehler bleiben die Daten unangetastet.

Achtung: Im Vollersatz-Modus werden die Tabellen der gewaehlten Kategorien vor dem Import geleert. Vorher unbedingt eine Sicherung anlegen.
