{# Generischer Edit-Modal-Helfer (DRY fuer Kontakt-, Liegenschafts-, Buchungsjahr-, Abrechnungsperioden-Modal). Body wird per HTMX-GET (X-From-Modal) geladen; das Formular submittet nativ via htmx (hx-post wird beim Oeffnen dynamisch gesetzt) — dadurch funktionieren Mehrfach-Checkboxen, und HX-Trigger-Antworten (204) werden von htmx selbst verarbeitet. setupEditModal({ formId, bodyId, modalId, // DOM-IDs des Skeletts openFn, // Name der globalen Open-Funktion (data-url-Button ruft sie) closeEvent, savedEvent, // HX-Trigger-Event-Namen vom Server savedHandler // optional: Name einer window-Funktion; sonst reload }) Modale, die denselben Dialog fuer Neuanlage UND Bearbeitung nutzen, koennen den aufrufenden Button mit data-modal-title="Neue Liegenschaft" versehen — der Titel-Text (Element mit [data-modal-title-text] im Modal-Header) wird dann beim Oeffnen aktualisiert. Buttons ohne dieses Attribut aendern den Titel nicht (bestehende reine Edit-Modale bleiben unberuehrt). #}
